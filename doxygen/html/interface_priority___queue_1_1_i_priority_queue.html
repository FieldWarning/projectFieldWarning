Detailed Description

The IPriorityQueue interface.

This is mainly here for purists, and in case I decide to add more implementations later. For speed purposes, it is actually recommended that you don't access the priority queue through this interface, since the JIT can (theoretically?) optimize method calls from concrete-types slightly better.
